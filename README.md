# OpenBlasphemousMap
A free and open-source interactive map for Blasphemous II.

Open Blasphemous Map é um mapa interativo gratuito e de código aberto para Blasphemous II. O projeto busca oferecer uma alternativa livre aos mapas comerciais, permitindo marcações ilimitadas, progresso salvo localmente, suporte às DLCs e contribuições da comunidade.
